# 🛡️ FaceShield AI – Identity Verification & Face Matching System

> **An AI-powered Identity Verification Platform that performs Face Detection, Face Recognition, Face Matching, Liveness Detection, and ID Verification using Deep Learning and Computer Vision.**

---

## 🚀 Overview

FaceShield AI is a production-ready Identity Verification System designed to automate digital KYC and user authentication using state-of-the-art Deep Learning and Computer Vision models.

The platform verifies a person's identity by matching a live selfie with government-issued identity documents such as Aadhaar Cards, PAN Cards, Passports, Driving Licenses, and Voter IDs.

Built using **Python, PyTorch, OpenCV, InsightFace, YOLOv8, FastAPI, React.js, and PostgreSQL**, FaceShield AI delivers secure, fast, and accurate identity verification for banking, fintech, healthcare, and enterprise applications.
